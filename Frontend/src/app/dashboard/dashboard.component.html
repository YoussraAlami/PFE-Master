SGA Metrics

The System Global Area (SGA) metrics represent different components of the Oracle database's shared memory area.

SGA Buffer Cache: Amount of memory used for caching data blocks.

SGA Fixed: Amount of memory allocated for fixed structures.

SGA Log Buffer: Size of the redo log buffer in memory.

SGA Shared Pool: Size of the shared pool memory area.

PGA Metrics

The Program Global Area (PGA) metrics represent different aspects of memory usage for each database session.

PGA Bound: Amount of PGA memory currently allocated.

PGA Allocated: Total PGA memory allocated.

PGA Freeable: PGA memory that can be freed.

PGA In Use: PGA memory currently in use.
